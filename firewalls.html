Firewalls

O firewall é basicamente um dispositivo que controla o tráfego que entra/sai de uma rede interna. Seu papel é impedir a saída de dados secretos bem como a entrada de usuários com intenções maliciosas.

Como podemos ver no esquema apresentado, o firewall funciona como uma barreira que impede alguns pacotes de serem encaminhados para os computadores da rede interna. O grande problema é que ele se trata apenas de uma barreira e, por isso, muitas vezes firewalls são utilizados em cada um dos computadores presentes na rede interna, além do(s) que toma(m) conta da rede inteira, para aumentar a segurança.
